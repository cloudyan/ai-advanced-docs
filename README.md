## 🎯 学习路径

### 推荐学习顺序

```
数学基础 → Python 基础 → 机器学习 → 深度学习 → Transformer → LLM 应用 → AI Agent
                                                            ↓
                                            系统优化 → 高级架构 → 研究 → 工程 → 创新
```

### 不同背景的学习者

| 背景 | 建议起点 | 重点 |
|------|---------|------|
| **数学/统计专业** | 阶段 3（机器学习） | 深度学习、Transformer |
| **计算机专业** | 阶段 2（Python） | 机器学习、深度学习 |
| **其他理工科** | 阶段 1（数学复习） | 全面学习 |
| **已有 AI 经验** | 阶段 5（Transformer） | 高级架构、研究 |
